# **App Name**: StudyBlossom

## Core Features:

- Study Goal Creation: Allow users to define their study goals, including the subject name, study time, and an associated icon.
- Feynman Method Recommendation: Recommend the Feynman method based on the user's defined study goal. It uses a content-based recommendation engine to recommend the method. Provide a clear rationale for the recommendation.
- Flashcard Creation: Use an AI tool to generate flashcards for the chosen study topic, allowing users to review key concepts efficiently, according to the Feynman method.
- Guided Feynman Steps (Text): Present a clear, step-by-step text guide to the Feynman method, including input fields for the user to record their explanations.
- Feynman Flashcards (Visual): Display flashcards that can be flipped or swiped, created in accordance with the Feynman technique, using the topic defined by the user.
- ElevenLabs Integration (Audio): Use the ElevenLabs API to offer an audio narration of the Feynman method steps, with selectable voices for enhanced engagement.
- Session Feedback: Gather feedback via a Likert scale survey to assess the usefulness of the recommendation and overall study session.

## Style Guidelines:

- Primary color: Soft pastel purple (#D8B4FE) for a calming and friendly atmosphere.
- Background color: Light pastel purple (#F5EEFE) for a clean, warm feel.
- Accent color: Light pastel pink (#F9A8D4) for highlighting interactive elements.
- Font: 'Poppins', a geometric sans-serif, for a modern and friendly aesthetic suitable for headlines and short paragraphs of text.
- Utilize custom rounded icons to represent study subjects and actions.
- Employ a clean, minimal layout with generous spacing to enhance readability.
- Incorporate subtle animations for button hovers and screen transitions to provide a smooth user experience.
